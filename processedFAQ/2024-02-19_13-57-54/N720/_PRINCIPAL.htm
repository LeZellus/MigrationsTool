FSE : Erreur F610H ou F650H à l'ouverture du logiciel

 

Diagnostic :

 

Message d'erreur : F610H ou F650H

 

Manipulation provoquant le message d'erreur : A l'ouverture du logiciel après la lecture CPS

 

Réponse :

 

API1.40.5.2

 

Vérifier le sesam.ini (dans c:\windows) que les chemins pointent bien sur programdata pour du Windows 7/8/8.1/10 et program files pour du Windows XP.  Vérifier que le sesam.ini ne soit pas présent dans un autre endroit du disque.

 

Vérifier aussi la présence du dossier sesam sur le poste dans programdata pour du Windows 7/8/8.1/10 et program files pour du Windows XP.

 

Si ça ne suffit pas ou si l'un des points n'est pas correct faire les correction ou réinstaller les API sesam vitale du logiciel concerné.

 

Si cela ne marche pas, essayer en réinstallant le répertoire Fonction Sesam Vitale (renommer le répertoire existant auparavant).

 

API 1.40.9.1

 

Si poste en XP et Crossway V20.06 et supérieures, il est possible que le sesam.ini ne pointe pas au bon endroit.

Le fichier est dans C:\ProgramData\santesocial\fsv\1.40.9\conf\

 

Sur un poste en windows 8 ou 10:

Tester l'accès au logiciel concerné en l'exécutant en tant qu'administrateur.

Si le problème est résolu ouvrir les droits sur le répertoire SanteSocial de Program Files et ProgramData, tester à nouveau pour confirmer que le problème venait des droits d'accès aux dossiers SanteSoecial.
