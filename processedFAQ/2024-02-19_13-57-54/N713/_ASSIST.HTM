f800Ch : caractéristique et/ou valeur Tcanal incorrecte

 

Diagnostic :

 

A la lecture carte CPS ou carte vitale ce qui indique que le port de communication sur lequel se trouve le lecteur dans le fichier Galss.ini est faux.

 

Réponse:

 

- pour un lecteur en USB, allez vérifier le n° de port sur lequel se trouve le lecteur dans le "Gestionnaire de périphérique" ligne "port COM et LPT"
- puis tapez dans Galss.ini dans "Démarrer/Exécuter ", vous trouverez dans le paragraphe "[CANAL1]", une ligne "index=" saisissez le numéro de port du lecteur
- fermez Galss.ini en enregistrant les modifications et refaite un test de lecture carte

- NB : pour les lecteurs en série index=1 ou 2.
