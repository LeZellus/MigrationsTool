E/S 32 pour ouvrir les résultats labos.

 

Diagnostic :

Le message apparaît lors de l'intregration des résultats labo par le menu Messagerie/Résultats Labo (Hprim).

 

Réponse:

 

Si ce message apparaît suivi d'autres messages qui indique qu'un fichier hprim ne peut pas etre ouvert, c'est qu'il y a une erreur dans le fichier reçu.-Supprimer le mauvais fichier manuellement qui se trouve dans : "C:\ProgramData\Cegedim\CLM\Hprim" ou "C:\Program Files (x86)\Cegedim\CLM\Hprim"en ayant auparavant fermé le logiciel.
